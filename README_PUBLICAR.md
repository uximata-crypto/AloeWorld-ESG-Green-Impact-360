# AloeWorld ESG & Green Impact 360

## Publicação no GitHub + Vercel

1. Crie um repositório vazio no GitHub.
2. Faça upload do ficheiro `index.html` desta pasta.
3. Confirme com **Commit changes**.
4. Na Vercel, clique em **Add New > Project**, escolha o repositório e faça **Deploy**.

Este ficheiro `index.html` é autónomo: o logótipo e o rodapé estão incorporados diretamente no ficheiro, não sendo necessária a pasta `assets`.

> Nota: esta versão ainda funciona em modo local para os dados do formulário. Para uma plataforma multiutilizador pública, será necessário ligar posteriormente ao Supabase.
